Introduction

PC Soundsets adds 30 completely new and original soundsets to the game, which you can use for your PC. The soundsets are all written and recorded by members of the modding community.

PC Soundsets is not a WeiDU mod, and can be installed before or after any other mod, without any compatibility issues (and you can of course uninstall at any time). For Windows users, the mod comes with a useful installer, which automatically downloads and installs the soundsets you choose.

Version 2 adds 8 new soundsets (3 female and 5 male) to complement the original 22. The OSX/Linux version also includes a Windows batch script in case you have problems with or don't want to use the installer.

Visit the website or forum for all the latest updates.

Installation

Please be aware that PC Soundsets is not a WeiDU mod! It will not be listed in your weidu.log! It can be installed before or after any other mod. It has no known incompatibilities with any other mods.

PC Installer (Recommended for most users)
PC Soundsets come with an automated installer. Download "shs#soundsets_v2.rar" from Spellhold Studios Download Center, and extract the installer ("shs#soundsets.exe") to any directory you wish.

Run "shs#soundsets.exe" and follow the instructions. You will be asked to select which soundsets you wish to install. You may choose how many or few as you wish.

The installer will then automatically download, extract and install the soundsets you selected. Make sure you are connected to the internet!

To uninstall in the future, run "uninstall.exe" from the shs#soundsets folder inside your game directory.

PC Batch script (Recommended for advanced users or if the installer doesn't work)
PC Soundsets also comes bundled together in a .zip file called "shs#soundsets_v2_full.zip." To install, please start by unzipping the contents to your Baldur's Gate II directory. You should have one new folder in your game directory called shs#soundsets. Inside it, you will find installation batch script for Windows ("install_win.bat"). Run the batch script to install. Please note that this will install all available soundsets.

To uninstall, please run the uninstallation batch script ("uninstall_win.bat") from the shs#soundsets folder inside your game directory.

OSX/Linux
PC Soundsets also comes bundled together in a .zip file called "shs#soundsets_v2_full.zip." To install, please start by unzipping the contents to your Baldur's Gate II directory. You should have one new folder in your game directory called shs#soundsets. Inside it, you will find installation shell scripts for both OSX and Linux ("install_osx.sh" and "install_linux.sh"). Run the appropriate script to install. Please note that this will install all available soundsets.

To uninstall, please run the uninstallation shell script ("uninstall_osx_linux.sh") from the shs#soundsets folder inside your game directory.
